# Contributing to A2I Curriculum

Thank you for your interest in contributing to the AI for Innovation curriculum!

## How to Contribute

### Reporting Issues

Use the issue templates to report:
- Curriculum feedback
- Bug reports
- Feature requests

### Suggesting Changes

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

### Curriculum Updates

For curriculum changes, please:
1. Use the curriculum update issue template
2. Provide clear rationale for changes
3. Include learning outcomes impact
4. Reference relevant pedagogy

## Review Process

All contributions are reviewed by:
- AI-assisted initial review
- Faculty curriculum committee
- Student feedback integration

## Style Guide

- Use clear, accessible language
- Include practical examples
- Reference academic sources where appropriate
- Follow markdown formatting standards

## Questions?

Open an issue or reach out to the curriculum team.
